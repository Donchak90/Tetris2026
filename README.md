# Фото-Тетрис — Telegram Mini App v2 (8-bit)

Изменения:
- 8-битный аркадный стиль.
- Пиксельный моноширинный шрифт.
- Прямоугольные панели и кнопки.
- Жёсткие рамки и тени.
- Таймер в буферной зоне: 3 секунды.
- Через 3 секунды фигура автоматически начинает падать.
- Игрок может запустить её раньше кнопкой или свайпом вниз.
